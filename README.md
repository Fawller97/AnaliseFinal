# AnaliseFinal
Temos todas as bibliotecas necessárias ja importadas para a execução dos códigos.

No primeiro ponto levantamnos o faturamento de todas as lojas limpando dados inconsistentes que não se categoriazavam como produtos removendo os mesmos para melhor análise

No segundo ponto levantamos as categorias de produtos mais e menos vendidos destacando os mais vendidos para questão de resultado mútuo entre as lojas

Em terceiro lugar procuramos ver a satisfação dos clientes em relação as lojas, isso serve como uma pesquisa se pulso, onde conseguimos medir questões como atendimento, qualidade de serviço e outras questões

Em quarto procuramos ver os itens mais vendidos, com isso conseguimos ter uma ideia até mesmo em relação de ticket médio algo que impacta diretamente no faturamento.

E finalizamos as análises de dados calculando o custo médio dos fretes das lojas, frete esse pago pelos cliente

--------

Acrescentamos uma analise com gráficos para facilitar os levantamentos de dados realizados e foi algo que ajudou a chegarmos a conclusão final incluida com o markdown.
